# Valerie De Leon - Court Reporter Invoicing System

A specialized invoicing and appeals tracking system built for court reporters.

## Features

*   **Invoicing**: Create, manage, and download PDF invoices.
*   **Appeals Board**: Trello-style tracking for appeals deadlines.
*   **Notifications**: Automated email and SMS reminders for deadlines.
*   **Dashboard**: Revenue tracking and statistics.
*   **Mobile Friendly**: Fully responsive design.

## Tech Stack

*   Next.js
*   Tailwind CSS
*   TypeScript
*   React

## Recent Updates

## Setup

1.  Clone repository
2.  `npm install`
3.  `npm run dev`
